Mission

Why we exist

Learning should be personal. Not a YouTube playlist. Not a generic curriculum. Not a one-size-fits-all course that assumes you're starting from zero or a genius.


StudyAI combines multi-agent AI systems, retrieval-augmented generation, and a deep understanding of learning psychology to build an experience that adapts to your knowledge level, schedule, goals, and learning style.


We believe that with the right AI system, anyone can master anything — given enough time and the right structure.

Technology

Built on solid foundations

⚗️
Flask + Python
Lightweight, high-performance web backend powering all API routes, authentication, and session management.
🦙
Groq + LLaMA
Ultra-fast inference with LLaMA 3.3 70B and Mixtral models via Groq's LPU hardware. Free and remarkably capable.
🧠
OpenAI GPT
Access GPT-4o and GPT-3.5 Turbo as an alternative provider for maximum model capability.
🔍
ChromaDB + HuggingFace
Local vector embeddings using sentence-transformers (all-MiniLM-L6-v2). No API key needed for RAG — runs fully on CPU.
🔗
LangChain
Document loading, text splitting, and vector store integration for the document Q&A pipeline.
🤿
Agno (phidata)
Multi-agent orchestration framework enabling specialized agent creation with DuckDuckGo web search capabilities.
